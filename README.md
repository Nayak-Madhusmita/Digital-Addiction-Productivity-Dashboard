# Digital-Addiction-Productivity-Dashboard
\📊 Smartphone Usage & Addiction Analysis Dashboard
📌 Project Overview

This project presents an interactive Tableau dashboard that analyzes smartphone usage patterns and their impact on key lifestyle and performance factors such as productivity, sleep, stress levels, focus, and burnout risk.

The goal of this project is to transform raw data into meaningful insights and help understand how excessive screen time affects daily life and well-being.

🎯 Objectives
Analyze smartphone usage behavior across different demographics
Identify relationships between screen time and productivity
Understand the impact of social media on focus and stress
Evaluate burnout risk based on usage patterns
Provide data-driven insights for better digital habits

🔍 Key Insights
Increased screen time leads to decreased productivity
Higher notification frequency increases stress levels
Better sleep is strongly linked to improved productivity
Social media usage negatively impacts focus score
Higher screen time is directly associated with increased burnout risk
🧠 What is Burnout Risk?

Burnout risk refers to a state of mental and physical exhaustion caused by prolonged stress and excessive workload or digital engagement.

📊 Findings from this project:
High burnout risk users have the highest screen time
Medium risk users show moderate usage patterns
Low risk users maintain controlled screen time

👉 This highlights the importance of maintaining a healthy balance between digital usage and personal well-being.
